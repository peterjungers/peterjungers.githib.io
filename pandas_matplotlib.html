As an exercise in using Python's pandas and Matplotlib libraries, I created the graphs below based on real-world data found at Kaggle. The data is taken from a 2023 live-streamed event on YouTube that aimed to determine coffee preferences of Americans via real-time tastings and surveys.

The complete Jupyter Notebook in which I worked can be viewed at my GitHub repository. Specific code examples, process notes, and the graphs themselves, can be viewed below.
