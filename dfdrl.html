Python: Flask and SQLAlchemy—DFDRL website

Various members of my family (only two of whom are children) participate in the Damon & Felix Die-Cast Racing League (DFDRL), where we construct tracks, keep statistics, and race Hot Wheels and Matchbox vehicles in events we participate in via Zoom sessions. Each member, in addition to managing their own racing team, has certain responsibilities to the league—I act as statistician and record keeper.

The website presented here was built using Python's Flask web application framework, which I first taught myself to use not long after my introductory Python course had ended. The database was built and queried using SQLAlchemy. I used Bootstrap for CSS styling, along with some custom styles—the images below illustrate desktop and tablet viewports. The project was also a good exercise in JavaScript, especially on the "Vehicles" page. The aesthetics of the site were based on my original Google Sheets statistics pages. The site was deployed at PythonAnywhere.
