About this site

The original version of this site was created in the winter of 2022 as part of an introductory course exploring various facets of Computer Information Systems. We spent two weeks learning about HTML and CSS. The site I created from scratch in that course can be viewed here. We required to include specific elements in the site (for example, we had to put a border around a div, or create an external hyperlink). The subject matter for the site, however, was of our choosing.

To help me learn more about the Bootstrap frontend framework, I decided, in the late fall of the same year, to use it to redesign the original site. The result is the site you are visiting now. I kept all the original subject matter, but all the HTML and custom CSS (which augments the Bootstrap code) is new.
