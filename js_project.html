JavaScript project

A couple of weeks into my introductory JavaScript course, I realized that I could turn the entire course into a project of its own. Though more advanced JavaScript can be found in more recent projects of mine, I wanted to include this old project in my portfolio because of how well it ultimately satisfied the goal I had initially set for myself: to create a fun, cohesive record of what I'd learned that term. We were often given the freedom to create any program as long as it met that week's lab requirements—not only did I enjoy thinking of new programs to create each week, but I also enjoyed trying to make new programs incorporate aspects of previous ones, both conceptually and aesthetically. More details about the project's genesis, as well as information for each lab, can be found on the project's landing page.
